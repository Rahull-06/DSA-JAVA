## 📂 What's Inside Each Folder

Each topic folder follows a consistent structure:

```
📁 Topic-Name/
├── 📄 Notes.md          # Concept summary & patterns
├── 📄 Problem1.java      # Clean solution with comments
├── 📄 Problem2.java
└── ...
```

Solutions include:
- ✍️ Problem statement (brief)
- 💡 Approach & intuition
- ⏱️ Time & Space complexity
- 🧪 Edge cases handled
